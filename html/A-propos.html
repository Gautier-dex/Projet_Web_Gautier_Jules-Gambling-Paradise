Présentation de la répartition du travail dans l'équipe de Gambling Paradise

Ce projet web a été réalisé par Jules Termoz & Gautier Dexemple. Le travail a été réparti équitablement et nous avons pu nous entraider. Nous tenons à signaler que toute utilisation d'IA importante pour la découverte de nouvelles fonctionnalités a été signalée à l'aide de commentaire (utilisation de l'IA pour debug certaines choses). Le but étant de rendre le site le plus agréable pour l'utilisateur.

CSS

Concernant le CSS nous nous sommes chacun occupé de nos pages respectives pour les modifications spécifiques d'ont il y avait besoin (mettre une image à droite et le texte à gauche, etc...), sauf pour la page nous rejoindre dont le CSS a été fait par Gautier. Gautier, s'est aussi occupé de faire le CSS pour le global (footer, police, couleur des titres, ...). Cependant le header a été réalisé par Jules.

Javascript

Pour le Javascript Jules s'est occupées des fonctions onload des pages strat et review, et des deux fonctions pour s'inscrire et se désinscrire. Gautier, s'est occupé de la fonction pour l'effet flou sur le tableau blackjack.
